# Tips

## Terminal

> How do I confirm what Ubuntu version I am using?

```bash
lsb_release -a
```

> How do I update available packages from the terminal?

```bash
sudo apt update
```

> How do I open the current directory in the file explorer from the terminal?

```bash
nautilus .
```
